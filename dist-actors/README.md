# CreatiVault Split Apify Actors

Generated by `npm run generate:actors`.

Each subdirectory is a self-contained Apify Actor that can be published separately.

- [CreatiVault Creator Search](./creator-search) - Search TikTok, Instagram, YouTube, and Twitter/X creators with follower, country, engagement, email, industry, and audience filters.
- [CreatiVault Lookalike Creator Finder](./lookalike-finder) - Find similar creators from a seed username or profile URL across TikTok, Instagram, and YouTube.
- [CreatiVault Video Search](./video-search) - Search short-form videos across TikTok, Instagram, and YouTube by hashtag, title, views, interaction rate, and publish date.
- [CreatiVault Creator Collection & Export](./collection-export) - Submit creator collection jobs by profile links, usernames, keywords, creator videos, or post videos, then fetch data or export xlsx/csv/html files.
- [CreatiVault Video Audit](./video-audit) - Submit TikTok, Instagram Reels, or YouTube Shorts for hook, script, storyboard, benchmark, and viral-factor analysis.
- [CreatiVault Outreach Email](./outreach-email) - Send creator outreach emails, check task status, inspect contact history, todo follow-ups, metrics, config, and upload attachments.
- [CreatiVault Files & Webhook Utilities](./files-webhook-utilities) - Utility actor for file delivery URL lookup and webhook connectivity verification.
- [CreatiVault OpenAPI Suite Internal](./openapi-suite-internal) - Internal all-in-one actor covering every CreatiVault OpenAPI operation, including debug/raw operations when enabled by environment variables.
